## 一、项目概述

**问乎** 是一款基于大语言模型的智能学习工具，采用苏格拉底式教学法，通过启发式提问引导学生自主思考。用户上传学习资料后，AI 自动提取知识点并以反问、追问方式驱动深度学习。

### 核心特色
- **Agent 自主教学**：AI 作为 Agent 自主决策 — 何时提问、提示、追问、结束
- **RAG 检索增强**：教学时检索资料原文，引用真实内容支撑提问
- **跨平台部署**：H5 + Android APK + 公网访问

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | uni-app (Vue 3 + Pinia + TS) | 跨端框架 |
| App 打包 | Capacitor 6 | H5 → Android APK |
| 后端 | Express.js + Node.js | API 服务 |
| 数据库 | MongoDB Atlas | 云数据库，数据持久化 |
| AI 模型 | DeepSeek API (deepseek-chat) | LLM 对话 + 知识点解析 |
| Agent 框架 | 自定义 Function Calling Loop | 轻量，无框架依赖 |
| RAG | 句子级 Chunking + 内存向量索引 | 混合检索（余弦相似度+BM25） |
| 公网隧道 | ngrok / localtunnel | 免费公网访问 |

## 三、已实现功能清单

### 用户系统
- 注册 / 登录 / JWT 认证
- 学习画像（知识点掌握度、错因记录）

### 资料管理
- 文件上传：PDF、Word(.docx/.doc)、图片、文本
- PDF 文本提取（pdf-parse）
- Word .docx 提取（mammoth）
- Word .doc 提取（二进制中文提取）
- AI 知识点自动解析（含 JSON 容错提取）

### Agent 教学核心（8个工具）

| 工具 | 功能 |
|------|------|
| `ask_question` | 苏格拉底式提问（基础/中等/高等） |
| `evaluate_answer` | 答案评估（正确/部分正确/错误 + 错因分类） |
| `provide_hint` | 分层提示（轻推/方向指引/详细引导） |
| `check_mastery` | 查询知识点掌握度 |
| `update_profile` | 更新学习画像 |
| `complete_session` | 结束会话 + 学习总结 |
| `search_knowledge` | RAG 检索资料原文 |
| `search_similar_points` | 跨资料知识点关联 |

### RAG 检索增强
- 上传后自动文档切分（句子边界感知）→ 向量化 → 入库
- Agent 教学时检索原文段落支撑提问
- 混合检索：语义相似度 + BM25 关键词

### 部署与发布
- MongoDB Atlas 云数据库（数据持久化）
- Capacitor Android APK 打包
- ngrok 公网隧道（真机远程访问）
- 国内 Gradle/Maven 镜像（加速构建）

## 五、关键修复记录

| 问题 | 原因 | 修复 |
|------|------|------|
| Agent 对话历史 tool 消息丢失 | MemoryManager 过滤掉 tool role | 保存 user/assistant/tool 全部消息 |
| tool_calls 配对被拆散 | trimContext 截断破坏结构 | 裁剪时检查配对完整性 |
| AI 知识点解析返回"知识点1/2/3" | DeepSeek 返回 JSON 带 markdown 包装 | `extractJSON()` 自动剥离 ```json``` |
| AI 解析时反问用户 | callAI 对所有请求注入苏格拉底 System Prompt | 移除 callAI 中的 System Prompt |
| Word .doc 文件无法解析 | mammoth 只支持 .docx | 新增 officeparser + 二进制中文提取 |
| 中文文件名乱码 | multer 编码问题 | `Buffer.from(name,'latin1').toString('utf8')` |
| 注册 API 失败（真机） | localtunnel 弹拦截页 + http→https | 换 ngrok + 加 bypass header + https |
| Gradle 下载失败（国内） | services.gradle.org 被墙 | 腾讯云镜像 + 阿里云 Maven 镜像 |
| Android 路径含中文拒绝构建 | Gradle 路径检查 | `android.overridePathCheck=true` |

## 六、环境配置

### .env 关键配置

```env
PORT=3000
MONGODB_URI=mongodb+srv://clay:<password>@cluster0.brnauce.mongodb.net/zhisixue?retryWrites=true&w=majority&appName=Cluster0
JWT_SECRET=zhisixue_secret_key_2024
DEEPSEEK_API_KEY=你的DeepSeek_API_Key
DEEPSEEK_API_URL=https://api.deepseek.com/v1/chat/completions
DEEPSEEK_MODEL=deepseek-chat
NODE_ENV=development
```

### 启动命令

```bash
# 后端
cd backend && npm start              # http://localhost:3000

# 前端开发
cd frontend && npm run dev:h5        # http://localhost:5173

# 前端构建 + Android 同步
cd frontend && npm run build:h5 && npx cap sync

# 公网隧道
ngrok http 3000 --domain=<你的域名>

# Android APK
Android Studio → Build → Build APK
```
